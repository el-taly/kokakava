## Списки

Чтобы добавить ненумерованные списки, необходимо пункты выделить (*).
Например, вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки, необходимо пункты пронумеровать:
1. Элемент 1
2. Элемент 2
3. Элемент 3
